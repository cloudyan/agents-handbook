# 从1加到100的计算方法

## 方法一：高斯求和公式（最高效）

数学家高斯在小时候就发现了这个巧妙的方法：

**公式：** 和 = (首项 + 末项) × 项数 ÷ 2

**应用到1加到100：**
- 首项 = 1
- 末项 = 100  
- 项数 = 100
- 和 = (1 + 100) × 100 ÷ 2 = 101 × 50 = **5050**

**原理：**
将数列配对：(1+100) + (2+99) + (3+98) + ... + (50+51)
每对的和都是101，共有50对，所以 101 × 50 = 5050

## 方法二：逐项相加（基础方法）

1 + 2 + 3 + 4 + ... + 98 + 99 + 100 = 5050

这种方法虽然直观，但效率很低，容易出错。

## 方法三：编程实现

```python
# Python示例
total = 0
for i in range(1, 101):
    total += i
print(total)  # 输出: 5050

# 或者使用内置函数
print(sum(range(1, 101)))  # 输出: 5050
```

## 总结

高斯求和公式是最优解，时间复杂度O(1)，而逐项相加的时间复杂度是O(n)。对于大数求和，公式法的优势更加明显。
